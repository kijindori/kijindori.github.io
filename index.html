플랫포머 게임

소스코드

    C++와 Win32 API 를 활용해서 구현했습니다.

발판을 밟아 점프하며 이동하는 형태의 게임입니다.

루프를 돌며 입력을 받고 받은 입력대로 캐릭터의 위치를 조정하도록 했습니다. 카메라가 캐릭터의 위치를 따라 이동하게 하였고, 각 기능별로 클래스를 만들었습니다.

콘텐츠 관리 크롬 확장 프로그램

    소스코드

    웹 문서를 찾아 읽을 때 유튜브 플레이리스트를 틀어두는 편인데, 유튜브 페이지로 돌아가서 다른 플레이리스트에 접근하는게 다소 불편해서 만들었습니다.

    찜한 OTT 콘텐츠를 다른 웹페이지를 보는 중에도 접근할 수 있도록 크롬 확장 프로그램으로 찜하기 기능을 구현했습니다.

    HTML, CSS, JavaScript 및 크롬 확장 프로그램 API를 활용했습니다.
